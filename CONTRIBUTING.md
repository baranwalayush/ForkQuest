### How to Contribute

We appreciate your interest in contributing to this repository. We welcome contributions in multiple areas — bug fixes, new features, documentation, and performance improvements.

### Code Contributions

1. Fork the repository
2. Clone the forked repository locally
- ```git clone https://github.com/your-user-name/ForkQuest.git```
3. Create a new branch for your change:
- ```git checkout -b feature/feature-name```
4. Make your changes and commit with a clear message:
5. Commit you changes
- ```git add .```
- ```git commit -m "commit message"```
6. Push your branch:
- ```git push origin feature/feature-name```
7. Create a Pull Request from your forked repository.

### Code Guidelines

To keep our codebase clean and maintainable, please follow these conventions:

- Use camelCase for variable and function names.
- Add comments where necessary.
- Use meaningful commit messages like (advised):

- - feat: for new features

- - fix: for bug fixes

- - refactor: for code improvements

- - docs: for documentation updates

- - Example:

- - - `git commit -m "feat: added filtering option by category"`

### Pull Request Guidelines

When opening a Pull Request (PR):

- Write a short title describing your change.
- Provide a clear description of what you did.
- Reference any related issue number.
- Ensure your code runs locally without breaking existing functionality.

### Do’s and Don’ts

#### Do:

- Test your changes before committing.
- Respect existing design and file structure.
- Ask questions in the discussion threads if unsure.

#### Don’t:

- Commit large dependencies unnecessarily.
- Push directly to the main branch.
- Submit copied code without understanding it.

### Issues

If you want to work on an issue:

1. Comment “I’d like to work on this” on the issue.

2. Wait for assignment confirmation.

3. Then start working on your PR.

### Need Help?

If you face any difficulty while contributing:
feel free to ask the maintainers

Remember: Every contribution, no matter how small, adds value to the project. We appreciate your effort and creativity!
